A repository documenting my journey of learning **Data Structures and Algorithms (DSA)** using **Java**.

## About

This repository contains my implementations, experiments, and practice programs while learning DSA from scratch.

Instead of only solving problems, I focus on understanding how data structures and algorithms work internally by implementing them myself.

The repository serves as a record of my learning journey, problem-solving approach, mistakes, improvements, and progress over time.

## Learning Philosophy

My approach to learning DSA:

1. Understand the concept.
2. Implement it from scratch.
3. Practice related problems.
4. Analyze Time and Space Complexity.
5. Revise regularly.
6. Improve the implementation over time.

The goal is not just to solve coding problems but to develop strong problem-solving and analytical thinking skills.
